# 第二课：数据类型与基本操作

## 基本数据类型
变量这个``“盒子”``里可以放不同类型的东西。在Python中，最基本的数据类型如下

**①字符串 (str)**：你刚才使用的 "tomori" 就是字符串，用于表示文本  
**②整数 (int)**：例如 10, -5, 0，用于表示没有小数部分的数字  
**③浮点数 (float)**：例如 3.14, 2.0, -0.5，用于表示有小数部分的数字  
这些类型可以进行基本的数学运算``（+, -, *, /）``。对于字符串，+ 操作符意味着拼接

**注意**  
```python
a = 10
b = 3
print(a / b)
>>>3.3333333335
```
10 / 3 的结果自动变成了一个浮点数 (3.333...)。这是Python的一个特性：除法运算 / 总是返回浮点数  
如果你想进行整数除法（只取商的整数部分），需要使用 // 操作符（例如 10 // 3 的结果是 3）  

# 第三课：逻辑控制 - if 语句

机器人做决策，程序控制流程，都离不开条件判断  
在Python中，我们使用 ``if``、``elif``、``else`` 语句来实现

你可以把它想象成一个**岔路口**  
**if** 是第一个路牌：如果满足某个条件，就走这条路。  
**elif** 是后续的路牌：否则，如果满足另一个条件，就走另一条路。  
**else** 是最后的路牌：如果以上条件都不满足，就走这条默认的路  

常用比较运算符
``==``（等于）  
``!=``（不等于）  
``> ``（大于）  
``< ``（小于）  
``>=``（大于等于）  
``<=``（小于等于）  

# 第四课：列表（List） - 数据的容器

在机器人和视觉中，我们经常需要处理一组数据，比如一帧图像中的所有检测框坐标、机器人关节的角度序列等  
Python中用于存储有序元素集合的最基本容器就是**列表（List）**  

## 基本操作

**创建列表**：my_list = [1, 2, 3, "hello"] （列表可以混合存放不同类型的数据，但通常我们存放同类型数据）    
**访问元素**：my_list[0] 获取第一个元素（结果是 1）  
**修改元素**：my_list[1] = 20 将第二个元素改为 20  
**获取长度**：len(my_list) 返回列表中元素的数量  

# 第五课：循环遍历列表 - for 循环

在上一课，我们打印了整个列表。但更多时候，我们需要逐个处理列表中的每一个元素。例如，对检测到的每个目标执行某种计算。这时就需要 for 循环  

for 循环就像一条流水线，自动将列表中的物品（元素）一个一个地取出来，交给你处理  
```python
for item in my_list:
    # 对 item 进行操作
    print(item)
    # item 是一个临时变量名，在每次循环中，它代表当前被取出的元素
    # my_list 是要遍历的列表
    # 注意 : 和缩进。缩进是Python定义代码块的唯一方式，循环体内所有语句必须缩进（通常用4个空格）
```

# 第六课：字典（Dictionary） - 键值对映射

列表通过``数字索引（0,1,2...）``来访问元素。但很多时候，我们想用更有意义的``标签（Key）``来存取数据。例如，在机器人配置中，我们想用 ``"max_speed"`` 而不是 ``config[0]`` 来获取最大速度  

Python的**字典（Dictionary）**就是为此而生。它是一个无序的 ``键(Key): 值(Value)`` 对集合  

**键（Key）** 必须是不可变的类型（通常是字符串或数字），且唯一  
**值（Value）** 可以是任何类型的数据  

## 基本操作

**创建字典**：``my_dict = {"name": "Robot", "speed": 1.5, "id": 101}``  
**访问值**：``my_dict["name"]`` （结果是 "Robot"）。如果键不存在会报错  
**安全访问**：``my_dict.get("name")`` 或 ``my_dict.get("color", "unknown")`` （如果键不存在，返回默认值）  
**修改/添加键值对**：``my_dict["speed"] = 2.0`` （修改已存在的键）或 ``my_dict["color"] = "red"`` （添加新的键值对）  

# 第七课：函数（Function） - 代码的模块化

当代码变得越来越复杂时，我们需要一种方式来封装可重用的逻辑块，这就是**函数（Function）**。在机器人编程中，函数用于定义特定的行为或计算，例如“计算到目标的距离”、“控制机器人移动一步”

函数就像一台专用的机器：你从入口（参数）投入原材料，机器内部进行处理，然后从出口（返回值）得到产品

## 定义函数
```python
def function_name(parameter1, parameter2):
    # 函数体
    result = parameter1 + parameter2
    return result
```
``def`` 是定义函数的关键字  
``function_name`` 是你给机器取的名字  
``(parameter1, parameter2)`` 是机器的入口，即参数  
``return`` 语句是机器的出口，将结果返回给调用者。如果没有return，函数默认返回None  

## 调用函数：
```python
output = function_name(10, 20) # output 将会是 30
```
